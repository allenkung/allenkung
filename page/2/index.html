ejs 中的 HTML tag 转义问题

2015-08-31
Web Development

最近在用 Express 开发公司网站，用了 i18n-node-2 这个多语言组件，需要将网站的文字部分都用 JSON 文件储存。遇到一个小问题，如果是带有 HTML 标签的内容，比如：

这里是<a href="#">链接</a>的文字内容

上面这段存到 JSON 数组里的话要转义一下：

"这里是<a href=\"#\">链接</a>的文字内容"

但是在 ejs 模板中输出的时候 HTML tag 总是 output 转义字符串（Escape Sequence）：

这里是&lt;a href=&quot;#&quot;&gt;链接&lt;/a&gt;的文字内容

查阅了 ejs 的 github README 文档中有如下介绍：

  • Unbuffered code for conditionals etc <% code %>
  • Escapes html by default with <%= code %>
  • Unescaped buffering with <%- code %>

发现想要 output HTML字符实体（Character Entities）的话只要使用 <%- code %>输出就可以啦！原来这么简单，本来还想是不是设计一个转义函数，瞬间觉得自己萌萌哒 ^_^

良心挂机头 Sigma 17-70/ƒ2.8-4 MACRO 入手

2015-07-08
Photography

七八年前读大学的时候就在 YY 这枚人物风景静物小品老少皆宜的副厂挂机头，当然到今天已经更新换代过几代产品了。此头良心之处在于 17-70 的黄金档焦段、最大 ƒ2.8 光圈、HSM 超声波对焦马达、OS 防帕金森，外加 1:2 微距。所有上述炫酷功能加起来总共只要 1998，只要 998（误）！

去年把柜子里的胶片相机陆陆续续都出掉了，也算玩过一些乱七八糟的器材。然后数码设备就只剩 05 年入的 Nikon D50 了（依然好用质量真好），镜头剩一枚 Sigma 10-20 残幅广角，主要用来拍摄风景和室内装潢。

今年年初的时候考虑换机身，主要是老机器毕竟十年了出片质量感觉锐度不够，不知是机身成像单元老化衰减还是今日和网上的图片相比要求变高了。Beta 同学推荐索大法 a7 全画幅微单，骚气十足，但是菜家镜头群价格偏高，不符合我只用平民单反的价值观。

然后转而考虑 Fuji 的复古旁轴系列，也是垂涎已久。x100 系列因为不能更换镜头所以不符合更换主力器材的出发点，x-pro1 现在价格公道、童叟无欺，而且镜头群个个良心。无奈见到真机后感觉体积略大，不符合我对小鸟依人的追求。

再则考虑到之前的 nikon 口 10-20 镜头，为了降低更新成本，所以在 D5500 刚上市的时候抄底购入上一代入门机 D5300。这两代变化不是很大，且 D5300 已经去处了低通滤镜（虽然我也不知道具体啥用）。最坑的是因为买的国行，所以 GPS 被阉割了……

正好手边有 Nikon 35/2 和广受好评的 Sigma 35/1.4，所以简单拍了一些样图供各位参考。大光圈依然是羡慕不来的，不过我最大的目的便是用 MACRO 来拍自己做的食物，敬请期待！

《归还照片》Mark Neville

2015-05-19
Photography / Lecture

上周末去民生美术馆听了 Mark Neville 的讲座《归还照片 Giving Photos Back》，Mark 是一位来自伦敦的摄影艺术家，正如大卫·康帕尼所说，「艺术家马克·内维尔重新构想了纪实摄影可以做到的事情以及应该做到的事情」。

讲座上，Mark 和大家 share 了过去的几个项目。Mark 的艺术作品是以一个个 project 的形式，所以每个 project 都有它的周期和阶段，Mark 展示了这些令人深思的作品从立项到完成和过程之间种种故事。从 Mark 的作品就可以看出，无疑他是一个非常优秀的 storyteller，在场的人们跟着 Mark 所描述的那些故事所包含的惊人清晰的细节，仿佛亲身参与到 Mark 的创作中。与此同时，也被 Mark 这些照片背后想要表述的苦苦寻求的隐喻所深深打动。在我看来，Mark 远超出了摄影师的范畴，他是一名令人尊敬的艺术家，相机只是他的工具。

我的上进心并未死去 — 哲学随笔

2015-04-13
Philosophy

「我的上进心并未死去」，这是小说《家》结尾巴金笔下的大哥觉新写给三弟觉慧的信上一行。本来这篇文章的题目应该更激列、或愤世嫉俗，比如「我的奋斗 — 哲学笔记」（希特勒梗）；或者是更诗意的类似「林中之路 — 哲学学习随笔」（海德格尔梗）。但随着年纪增大，年少时读过的觉新 — 这个既向腐旧的礼教妥协，同时又支持弟弟决绝的革命理想，左右摇摆、痛苦不已的两难角色 — 越来越生动和清晰。这个处境是不是很像萨特当年寻找的第三条路？回忆读哲学的过程，一路走来，就是抱着觉新的这种心态，一个塞林格笔下的麦田守望者。

每日时间分配和效率的研究探讨

2015-02-09

年初列了许多需要完成的目标，最近在思考从何着手，突然发现自己一直以来都忽略了一个显而易见的事实：无论工作狂还是拖延症晚期病患，一天只有 24 小时。想明白了这个问题，再算一算每天的闲余时间，不禁有股紧迫感涌上心来（当然拖延症该来的时候还是不可阻挡地来）。

工作日去掉「睡眠」和「交通」、「工作」固定占用的时间，还要算上吃饭（一日三餐平均每顿 1 小时，再强行挤进洗漱的时间），每天只剩下可怜的 4 小时可供自己调用的业余时间了。周末如果不睡懒觉的话，也就多出 8 x 2 小时，这样一周满打满算也就 ( 4 x 7 ) + ( 8 x 2 ) = 44 小时。如果按照「一万小时定律[^1]」来算，那么所有的业余时间用来做同一件事情，四年半以后就可以成为这个领域的专家。如果同时有多个业余爱好，那么需要花上十年或者更久才行，强烈感觉到再不行动就老了！

以前总感觉还有整年的时间不争朝夕，其实就算完全放弃社交、发呆装傻的时间，一年也只有 2280 小时 / 95 天属于自己的时间（自由时间伤不起）。

写到这里，觉得有必要把年初定的目标们重新计划一下才能确保年尾能顺利交差啊！

[^1]: 人们眼中的天才之所以卓越非凡，并非天资超人一等，而是付出了持续不断的努力。1万小时的锤炼是任何人从平凡变成超凡的必要条件（要成为某个领域的专家，需要 10000 小时）。 —— 格拉德威尔《异类》
